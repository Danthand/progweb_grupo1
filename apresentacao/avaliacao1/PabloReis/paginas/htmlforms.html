The input Element The input element is the most important form element. The input element is displayed in several ways, depending on the type attribute. Here are some examples: Type Description input type="text" Defines a single-line text input field input type="radio" Defines a radio button (for selecting one of many choices) input type="submit" Defines a submit button (for submitting the form) You will learn a lot more about input types later in this tutorial.
